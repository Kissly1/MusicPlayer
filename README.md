# 🎵 iOS Music Player

![Swift](https://img.shields.io/badge/Swift-5.0-orange.svg)
![Framework](https://img.shields.io/badge/AVFoundation-Apple-lightgrey.svg)

A custom music player application for iOS that allows users to play, pause, and navigate through tracks. This project focuses on integrating native Apple frameworks for audio playback.


## ✨ Features
* Audio playback controls: Play, Pause, Next, Previous.
* Volume control and track progress tracking.
* Custom UI designed using UIKit and Storyboards.

## 🛠 Tech Stack & Frameworks
* **Language:** Swift
* **UI:** UIKit (Storyboards & AutoLayout)
* **Frameworks:** `AVFoundation` (for audio playback and state management)
